Java异常处理的一个例子

本文记录了Java异常处理情况的一个例子：

  1. case1：

    try代码：throw new Exception();

    打印语句：“catch” “finally”

  2. case2:

    try 代码： 空

    打印语句： “finally”

    结 论： try代码正常结束，finally语句块被调用

  3. case3:

    try 代码： return;

    打印语句：“finally”

    结 论： try代码return，finally语句块被调用

一般情况下，调用WebView.loadUrl(url)方法即可加载需要展示的Html页面了，如果需要传参数的话，我们也可以把参数拼在url后面(像这样http://www.baidu.com/s?word=WebView)，一并传给服务端。但有时候情况比较特殊，比方说，请求页面的时候我们还想通过post方式传一些参数过去，这个时候直接调用上述方法就不行了。这个时候该怎么办呢？
